# **Template Engines**

We would like to search online for the latest, most widely adopted standards and open-source (strongly preferred) tools/libraries for such templated document processing, generation, and verification

  * It is **acceptable** if it only supports/implements a single/few most standard formats for the generated output, as long as there are other open-source tools that can be chained seemlessly to convert input formats of the template/content, AND if the ones that are supported are the standard/text-only ones that are easier to track in git/convert to other formats (such as MarkDown \-\> PDF)
